使用指南

保存与提交

当前图片修改后开启自动保存会自动保存；保存会写入 JSON 和同名 YOLO txt。

状态文件会自动生成在当前文件夹中。

导入与整理

命名混乱或目录嵌套时，可先发给项目团队整理。

图像显示

亮度和对比度只影响当前显示，不改变图片、关键点坐标或导出文件；顶部滑块和右侧拖动板同步，设置会写入当前标注文件。

当前 schema 为每侧 12 点，共 24 点；#7 为股骨颈轴中心远端，#12 为股骨颈轴中心近端。自动识别时 #12 暂用同侧 #3 和 #7 的中点估计，可人工调整；#10/#11 默认可选，不计入完成缺失。
