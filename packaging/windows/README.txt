Hip 22 Annotation Tool - 髋关节 X 光标注工具
================================================

快速上手
--------
1. 解压整个 ZIP，不要只复制 exe。
2. 双击 Hip22AnnotationTool.exe。
3. 等待几秒，浏览器会自动打开 http://127.0.0.1:8010/。
4. 在界面中点击“导入文件夹”，选择已经整理好的待标注图片文件夹。
5. 等自动识别完成后，逐张检查并拖拽关键点，必要时手工放点或删除点。
6. 点击“保存”。完成后把整个标注文件夹发回给我们。

完整图文流程见同目录下的 Hospital-User-Guide-zh.md。

如果浏览器没有自动打开
--------------------
手动在浏览器地址栏输入 http://127.0.0.1:8010/。

如果 Windows 出现安全提示
------------------------
这是内部未签名程序的常见提示。点击“更多信息”，再点击“仍要运行”。

数据保存在哪里
--------------
默认设置、日志和临时工作区在：
  %LOCALAPPDATA%\Hip22AnnotationTool\

导入某个图片文件夹后，标注 JSON 会保存到该文件夹的 annotations 子目录，YOLO txt 会保存到图片同目录。
医生提交时不需要单独导出，把整个标注完成的文件夹发回即可。

排查问题
--------
如果程序闪退或打不开，请双击 Run-Hip22.bat。它会把日志写到：
  %LOCALAPPDATA%\Hip22AnnotationTool\logs\run-<时间>.log

把这个日志文件发给我们排查。
