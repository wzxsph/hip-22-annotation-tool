# Hip 24 髋关节 X 光标注工具使用说明

本工具用于辅助标注髋关节 X 光片上的 24 个关键点。当前默认每侧 #10/#11 为可选点，不计入完成缺失；自动识别结果只作为初始参考，请医生逐张复核并修正。

v0.3.2 预发布版额外支持人工确认工作流（关键点确认 + 沈通线确认）、11 条默认解剖引导连线、DICOM 导入、增强预览、ROI 辅助识别、手机拍摄片的类扫描四角校正、Shenton 曲线研究标注和实时测量指标，并修复 Windows 发布包中回收站路径的跨平台兼容性。Shenton 连续性候选、AI/Tonnis 角、Sharp 角、CE 角、颈干角和髋臼深度仅用于研究记录和医生复核辅助，不作为临床诊断结论。

## 1. 准备数据集

1. 在电脑上新建一个文件夹，例如 `D:\hip22_annotation\待标注病例`。
2. 把本次需要标注的所有 X 光图片放进这个文件夹。
3. 图片可以是 `.jpg`、`.jpeg`、`.png`、`.bmp`、`.tif`、`.tiff`、`.webp`、`.dcm` 或 `.dicom`。普通单帧灰度 DICOM 会自动读取；压缩或私有格式无法读取时会在界面提示。
4. 尽量不要把无关文件放入该文件夹。

如果图片命名混乱、分散在很多子文件夹、无法判断哪些应该标注，或者系统无法导入，请先把要标注的照片发给我们。我们可以帮忙整理成统一文件夹，再发回给医院继续标注。

## 2. 启动工具

1. 解压我们提供的 `Hip22AnnotationTool-v...-win64-cpu.zip`。
2. 打开解压后的 `Hip22AnnotationTool` 文件夹。
3. 双击 `Hip22AnnotationTool.exe`。
4. 等待几秒，浏览器会自动打开标注界面。

如果浏览器没有自动打开，请手动访问 `http://127.0.0.1:8010/`。如果黑色窗口关闭，工具也会退出。

## 3. 导入文件夹并自动识别

1. 点击顶部的“导入文件夹”。
2. 点击“选择”，选中准备好的图片文件夹。
3. 点击“导入”。
4. 导入后，系统会列出文件夹中的图片，并在后台使用增强图自动识别关键点。

自动识别是辅助功能。如果某些图片识别不出来，可以继续手工标注；如果大批量识别异常，请把日志和图片文件夹情况反馈给我们。当前 #12 “股骨颈轴中心近端”会暂用同侧 #3 和 #7 的中点自动估计，医生可继续拖动修正。

如果某一张图片没有识别出点，或想对当前图片重新尝试，可以先打开这张图片，再点击顶部“自动识别”。该操作会重新运行模型，保留已经人工修改过的点；如果模型只能给出少量候选点，也会显示出来供医生拖拽复核。若仍没有点，请继续手工标注或把图片发给我们排查。

打开图片后默认显示增强图，便于观察泪滴或骨性边界；如需对照原片，可以点击“原图”。增强预览不会改动原图和坐标。导入文件夹后的后台识别默认使用增强图；也可以点击“增强识别”让系统用同一套增强逻辑重试当前图片，仍会保留人工修改点。增强图和 DICOM 显示图会缓存在本机工具数据目录中，反复切换图片时不需要每次重新计算。
如果图片中包含手机拍摄背景、桌面边框或其他杂乱区域，可以点击左侧“ROI”，在画布上拖拽框出真正需要关注的 X 光区域。之后点击“自动识别”或“增强识别”时，系统会优先在 ROI 内重试识别；ROI 不会裁掉原图，保存结果仍然使用原图坐标。
“适配”按钮用于把当前图片自动缩放并居中到画布中；在 ROI 工具下，如果已经画了 ROI，可以使用“适配 ROI”快速放大到关注区域。

如果是患者手机拍摄的 X 光片，且能看清片子四个边角，可以点击左侧“扫描”，按左上、右上、右下、左下顺序点击四个角。之后点击“自动识别”或“增强识别”时，系统会先把这一区域透视校正成接近扫描件的视图再识别，并把识别点映射回原图坐标。扫描四角不会修改原图，也不会单独生成需要提交的派生图片。

导入后，右侧“文件夹进度”会显示未标注、待复核、修改中、已完成的数量。工具也会在图片文件夹里自动生成：

- `HIP22_STATUS_DONE_..._TODO_....txt`：一眼查看完成和待处理数量。
- `HIP22_status_report.html`：逐张查看状态。
- `HIP22_status_report.csv`：团队整理时使用。
- `HIP22_SUBMISSION_README.txt`：提交说明。

如果导入时发现嵌套目录、重复文件名、不支持文件或疑似混乱命名，界面会提示。遇到这些情况可以先把照片发给我们整理。

## 4. 修改标注点

1. 点击底部缩略图打开某一张图片。
2. 鼠标拖拽图上的点，可以调整点位。
3. 左侧可以选择具体点位；右键图片也可以快速放置某个点。
4. 鼠标滚轮可以放大缩小，按住空格拖动可以平移图片。
5. 如果某个点确实不可见，可以选中该点后按 `Delete` 标为缺失。

拖拽后的点会变为人工修改结果。请重点确认自动识别点是否落在正确的临床位置。

## 4.1 Shenton 曲线研究标注（如需）

1. 点击左侧工具区的 “Shenton”。
2. 选择图像左侧或图像右侧。
3. 先选择“闭孔上缘”，沿闭孔上缘曲线点击至少 3 个点；边界越长、越弯时可以多标一些点。
4. 再选择“股骨颈内下缘”，沿股骨颈内下缘曲线点击至少 3 个点；点越多越贴合曲线。
5. 如点位不准，可以拖拽曲线点；也可以撤销最后一点或清空当前段重画。
6. 右侧会显示两段曲线最近端点的 gap；如 DICOM 有有效标尺，会同步显示毫米距离。

该功能目前用于 Shenton 线标注方案研究和医生复核记录，不输出临床诊断结论。最终连续/不连续仍以医生判断为准。

## 4.2 实时测量指标（如需）

右侧“Shenton 与测量”区域会根据当前 24 个点位自动刷新 AI/Tonnis 角、Sharp 角、CE 角、颈干角和髋臼深度。若当前图片是带有效 PixelSpacing 的 DICOM，髋臼深度会同步显示毫米值；否则只显示像素距离。当前 24 点 schema 没有单独的泪滴外缘点，髋臼深度暂以 `teardrop_lower` 作为代理点计算，结果需要医生复核。

## 5. 确认完成

标注完关键点后，点击右侧"确认关键点完成"按钮，系统会将该图标记为关键点已完成。标注完沈通线后，展开"Shenton 与测量"面板，点击"确认沈通线完成"。两者都确认后，该图在进度中显示为"完成"。

右侧面板有默认浅红色解剖引导连线（髋臼、股骨头、股骨干、跨中线）。#10/#11 点在画布上默认隐藏、默认可选，可在"高级设置 > 显示"中打开后补充标注。

## 6. 保存

点击顶部“保存”。工具默认也会自动保存，并且切换图片前会先保存当前图片的修改。

如果误导入了不需要标注的图片，可以点击“删除本图”。工具会把图片、同名 JSON 标注和 YOLO txt 移动到图片同目录的 `trash` 文件夹，并从当前列表中移除；如需恢复，可以手动从 `trash` 移回原位置。

保存后，标注结果会写入导入的图片文件夹：

- `annotations` 子文件夹：完整 JSON 标注结果。
- 图片同目录下的同名 `.txt`：YOLO 标注结果。

## 7. 提交给我们

完成全部图片后，请把整个标注完成的文件夹发给我们，不需要单独导出。

提交前可看右侧“提交检查”：如果仍有“未标注、待复核、修改中”的图片，请继续处理；全部完成后再发送整个文件夹。

完整流程是：

准备数据集 -> 导入文件夹 -> 自动识别 -> 拖拽修改标注点 -> 保存 -> 将整个文件夹发送给我们

## 常见问题

### 命名太乱怎么办？

可以先发给我们整理。我们会统一文件名、检查无法读取的图片，并保留原始文件名映射。

### 自动识别没有点怎么办？

可以先打开该图片，点击顶部“自动识别”重试。工具会尝试多种识别参数；如果仍然没有识别到点，未识别的点会保持缺失，需要人工放点。软件不会再用模板点自动补齐缺失点；如果多张图都没有自动点，请把日志和图片情况发给我们。

### 程序打不开或闪退怎么办？

双击 `Run-Hip22.bat`，它会生成日志文件。请把日志发给我们排查。

### 保存后应该发哪些文件？

发整个导入后的文件夹。不要只发 `annotations`，也不要只发 `.txt`。
