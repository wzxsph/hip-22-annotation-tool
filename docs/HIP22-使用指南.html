HIP22 髋关节 X 光标注工具使用指南

当前版本用于标注每侧 12 个关键点，共 24 个关键点。#12 会在自动识别后暂用同侧 #3 和 #7 的中点估计，医生可继续人工调整；#10/#11 默认可选，不计入完成缺失。自动识别结果只作为初始参考，最终结果以人工复核、保存后的 JSON 和同名 YOLO txt 为准。

1. 导入与工作区

  1. 点击“导入文件夹”，选择包含原始图片的病例目录。
  2. 打开缩略图后，默认先显示原图；增强图只作为查看和辅助识别，不覆盖原始图片。
  3. 命名混乱或目录嵌套时，可先把文件夹发给项目团队整理。

2. 标注流程

  1. 逐张检查自动识别点位，拖动错误点，缺失点用“放点”补齐。
  2. 完成后点击“确认关键点完成”；Shenton 曲线完成后点击“确认沈通线完成”。
  3. 保存会写入 annotations 目录下的 JSON，同时在图片同目录写入 YOLO txt。
  4. 若开启“自动保存”，当前图片修改后会自动保存；亮度/对比度设置也会写入 JSON 的 display_settings。

4. 亮度与对比度

  • 顶部滑块适合精确调整；右侧“显示调节”面板适合快速拖动调整。
  • 在右侧调节板中按住鼠标左键拖动：向左/右调对比度，向上/下调亮度。
  • 这些参数只影响显示，不改变原图像素和关键点坐标；保存后会进入 JSON，供后期归一化参考。

5. 删除、还原与缩略图

  • 底部缩略图可勾选多张，点击“删除所选”批量删除原图片及同名标注文件。
  • “还原图片”可从当前工作区 trash 文件夹中恢复误删图片。
  • 当前图片会在缩略图条中以红色粗框高亮。
  • 已经画好的 Shenton 曲线点可在 Shenton 工具或选择工具下直接拖动调整。

7. 完成提交

工作区会生成 HIP22_STATUS_DONE_..._TODO_....txt、HIP22_status_report.html、HIP22_status_report.csv 和 HIP22_SUBMISSION_README.txt。提交前请打开 HTML 报告确认未完成图片。

自动识别模型目前仍只直接输出原有每侧 11 个模型点；#12 “股骨颈轴中心近端”由同侧 #3 和 #7 的中点自动估计，仍需医生复核。
